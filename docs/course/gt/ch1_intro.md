# 第一章 导论

## 基本概念

**博弈**：个人或组织的决策行为，特别是有<u>策略互动和利益依存特征的决策行为</u>。

**博弈论**：专门研究具有<u>策略互动和利益依存特征的决策行为</u>的理论。

**博弈方**：博弈中<u>独立决策、独立承担博弈结果</u>的个人或组织。

**策略**：博弈中各博弈方的决策内容，即规定每个博弈方可以选择的行为或经济活动的水平、量值等。

**得益**：博弈方从博弈中获得的利益，它是博弈方<u>决策行为的主要依据</u>。

**得益矩阵**：一种通过<u>数组矩阵</u>形式来直观表示博弈问题的方法。矩阵的行和列分别代表不同博弈方的可选策略，矩阵内的每个数组表示对应行列代表的双方策略组合下的各自得益。

**扩展形**：一种通过图形化方式（包含选择节点、代表可能选择的行动路径线段，以及终端处的得益数组）来表示博弈问题的方法。能够形象、直观地反映出博弈的策略路径，以及博弈过程中各方选择行为的先后次序。

## 博弈理论结构

### 分类总览

按**博弈方的数量**：单人博弈、双人博弈和多人博弈

按**博弈方策略的数量**：有限博弈和无限博弈

按**博弈中的得益情况**：零和博弈、常和博弈和变和博弈

按**博弈过程**：静态博弈、动态博弈和重复博弈（有限重复博弈/无限重复博弈）

按**信息结构**：完全信息博弈和不完全信息博弈，完美信息动态博弈和不完美信息动态博弈

按**博弈方的理性程度**：可以分为完全理性博弈和有限理性博弈

按**博弈方式差别**：非合作博弈和合作博弈

### 按博弈方的数量分类

单人博弈 / 双人博弈 / 多人博弈

### 按博弈方策略的数量分类

- 有限博弈：一个博弈中，每个博弈方可选择的策略数量都是有限的。
- 无限博弈：一个博弈中，<u>至少有一个</u>博弈方拥有无限多种可选择的策略。

### 按博弈中的得益情况分类

- 零和博弈：无论博弈结果如何，所有博弈方的总得益始终为0的博弈。
- 常和博弈：无论博弈结果如何，所有博弈方的总得益始终为<u>某一非零常数</u>的博弈。
- 变和博弈：<u>在不同的博弈结果下</u>，博弈方的总得益也不同的博弈。

### 按博弈过程分类

- 静态博弈：所有博弈方同时<u>或可看作同时</u>进行决策的博弈。
- 动态博弈：各博弈方的选择和行动有先后次序，即无法看作是同时进行决策的博弈。
- 重复博弈：重复进行某个特定博弈所构成的博弈过程。
  - 有限次重复博弈：重复一定次数后肯定会结束的重复博弈（至少2次）。
  - 无限次重复博弈：没有预期的结束时间，各博弈方认为重复过程会一直进行下去的重复博弈。

### 按信息结构分类

- 完全信息博弈：各博弈方完全了解所有博弈方在各种情况下的得益情况的博弈。
- 不完全信息博弈：至少部分博弈方<u>不完全了解</u>其他博弈方得益情况的博弈。
- 完美信息动态博弈：动态博弈中，所有博弈方对各自行动之前的博弈进程和历史有完美知识的博弈。
- 不完美信息动态博弈：动态博弈中，至少一个博弈方在轮到行动时<u>不完全了解</u>此前全部博弈进程的博弈。

### 按博弈方的理性程度分类

- 完全理性博弈：假设博弈方是完美的理性人，严格按照自身利益或效用最大化原则进行决策的博弈。
- 有限理性博弈：博弈方在预见性、判断分析或决策能力上存在局限的博弈。

### 按博弈方式分类

- 非合作博弈：博弈方个体独立进行决策和行为，博弈本身不以博弈方的合作为前提。
- 合作博弈：<u>博弈方式</u>必须包含博弈方之间的协调合作的博弈。

## 例题

**1. 什么是博弈？博弈论的主要研究内容是什么？**

博弈可以用下述方式定义：“博弈即一些个人、队组或其他组织，面对一定的环境条件，在一定的规则下，同时或先后，一次或多次，从各自允许选择的行为或策略中进行选择并加以实施，各自取得相应结果的过程”。一个博弈必须包含博弈方、策略空间、博弈的次序和得益这几个基本的方面。信息结构、博弈方的行为逻辑和理性层次等其实也是博弈问题隐含或者需要明确的内容。博弈论是系统研究可以用上述方法定义的各种博弈问题，寻求在各博弈方具有充分或者有限理性、能力的条件下，合理的策略选择和合理选择策略时博弈的结果，并分析这些结果的经济意义、效率意义的理论和方法。

**2. 设定一个博弈模型必须确定哪几个方面？**

1. **博弈方**，即博弈中进行决策并承担结果的参与者；
2. **策略（空间）**，即博弈方选择的内容，可以是方向、取舍选择，也可以是连续的数量水平等；
3. **得益或得益函数**，即博弈方行为、策略选择的相应后果、结果，必须是数量或者能够折算成数量；
4. **博弈次序**，即博弈方行为、选择的先后次序或者重复次数等；
5. **信息结构**，即博弈方相互对其他博弈方行为或最终利益的了解程度；（完全/完美信息）
6. **行为逻辑和理性程度**，即博弈方是依据个体理性还是集体理性行为，以及理性的程度等。（完美理性）

**3. 博弈有哪些分类方法，有哪些主要的类型？**

1. 根据**博弈方的行为逻辑**：非合作博弈和合作博弈；
2. 根据**博弈方的理性层次**：完全理性博弈和有限理性博弈；
3. 根据**博弈过程**：静态博弈、动态博弈和重复博弈；
4. 根据**信息结构**：完全信息静态博弈、不完全信息静态博弈、完全且完美信息动态博弈、完全但不完美信息动态博弈和不完全信息动态博弈；
5. 根据**得益特征**：零和博弈、常和博弈和变和博弈；
6. 根据**博弈方的数量**：单人博弈、双人博弈和多人博弈；
7. 根据**博弈方策略的数量**：有限博弈和无限博弈。
